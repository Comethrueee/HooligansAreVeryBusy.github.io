git常用命令

总结下git版本管理

  1. git常用命令

    1. git clone #克隆项目
    2. git push #推送
    3. git add #添加
    4. git commit #提交
    5. git checkout #切换分支
    6. git pull #拉取

  2. 要用记不住的命令

  • git reset –soft HEAD^ # 取消commit,保留修改的文件
  • git reset –hard HEAD^ # 取消commit,不保留修改的文件
  • git checkout -b shen-dev(本地分支名) origin/release_dev(远程分支名) #切换远程分支
  • git checkout . #恢复工作树文件
